# App Assets
These files are generated from BizBil brand assets already stored in `apps/web/public`.
Regenerate them with `node apps/mobile/scripts/generate-placeholder-icons.js` after any brand asset update.
- `icon.png`: 1024x1024 main app icon
- `adaptive-icon.png`: 1024x1024 Android adaptive foreground asset
- `splash.png`: 1284x2778 splash image
- `notification-icon.png`: 96x96 Android notification icon
